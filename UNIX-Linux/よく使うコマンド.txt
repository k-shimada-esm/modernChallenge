1.cd    [change directory]
    cd ~    ：/home/ユーザー名  (cd $HOME)
    cd /    ：LinuxのインストールされたOS内での一番上のディレクトリに移動
    cd ../  ：いまいるディレクトリーの、ひとつ上のディレクトリーに移動

2.ls    [list segments]
    ls      ：カレントディレクトリの中身。存在確認で使用
    ls -l   ：より詳細な情報を表示（パーミッション、ファイルサイズ、作成者、作成された時間、最後に変更された時間など）
    ls -lh  ：ファイルサイズを KB, MB などで表示
    ls -a   ：ディレクトリ内にある全てのファイルを表示（. ..）
    ls -R   ：ディレクトリを再帰的に下降してファイルを表示

3.pwd   [Print Working Directory]
    現在のディレクトリを返す

4.sudo  [switch user do]
    sudoのあとに管理者権限で実行したいコマンドを書く。
        sudo コマンド           ：スーパーユーザとして実行
        sudo -u ユーザ コマンド ：他のユーザとして実行

5.rm    [remove Directory]
    指定したファイルを消去。
    一度消去したファイルは取り戻せないので注意！！
    「-i」を付ける癖。
        rm -i ファイル名  ：削除前に確認する

6.cp
    ファイルをコピーする。
    コピー先のファイルを無条件に上書きするので注意！！
    「-i」を付ける癖。
        cp -i 元ファイル名 先ファイル名  ：上書き前に確認する

7.mv
    指定されたファイルを指定された場所に移動する。
    移動先のファイルを無条件に上書きするので注意！！
    「-i」を付ける癖。
        mv -i 元ファイル名 先ファイル名  ：上書き前に確認する
